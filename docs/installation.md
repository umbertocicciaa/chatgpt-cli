# Installation

## Prerequisites

- Go 1.19 or later
- An OpenAI API key

## Install with Go

```bash
go install github.com/umbertocicciaa/chatgpt-cli@latest
```

## Build from Source

```bash
git clone https://github.com/umbertocicciaa/chatgpt-cli.git
cd chatgpt-cli
go build -o chatgpt-cli main.go
```

## Download Pre-built Binaries

Pre-built binaries are available on the [Releases](https://github.com/umbertocicciaa/chatgpt-cli/releases) page for the following platforms:

| Platform       | Architecture |
|----------------|-------------|
| Linux          | amd64, arm64 |
| macOS (Darwin) | amd64, arm64 |
| Windows        | amd64, arm64 |
