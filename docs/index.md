# ChatGPT CLI

A modern, extensible command-line interface for ChatGPT with subcommand support, configuration management, and comprehensive logging.

## Features

- Interactive chat sessions with ChatGPT
- Subcommand-based architecture
- Configuration management
- Comprehensive logging
- Cross-platform support (Linux, macOS, Windows)

## Quick Start

```bash
# Install
go install github.com/umbertocicciaa/chatgpt-cli@latest

# Set your API key
export OPENAI_API_KEY="your-api-key"

# Start chatting
chatgpt-cli chat "Hello, how are you?"
```
