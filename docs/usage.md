# Usage

## Basic Usage

```bash
# Set your API key
export OPENAI_API_KEY="your-api-key"

# Send a message
chatgpt-cli chat "Hello, how are you?"
```

## Available Commands

Run `chatgpt-cli --help` to see all available commands and options.

## Examples

### Simple Chat

```bash
chatgpt-cli chat "Explain quantum computing in simple terms"
```

### Using Different Models

```bash
chatgpt-cli chat --model gpt-4 "Write a haiku about programming"
```

For more details, see the [Configuration](configuration.md) page.
